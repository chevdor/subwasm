Compare 2 runtimes after converting them to `ReducedRuntime`s.

You must pass exactly 2 runtimes.

Usage: subwasm diff [OPTIONS] <RUNTIME_1> <RUNTIME_2>

Arguments:
  <RUNTIME_1>
          Reference runtime

  <RUNTIME_2>
          Second runtime

Options:
  -j, --json
          Output as json

  -n, --no-color
          [env: NO_COLOR=]

  -q, --quiet
          Less output

  -h, --help
          Print help (see a summary with '-h')
